# gmath

Gmath (written as gmath) library is solely created as an exercise for following things:

* CI (Github Actions)
* C++ (templates, optimization, unit testing etc.)
* Game maths (vectors, matrices, quaternions, euler angles etc.)
* CMake
* SIMD (just maybe?)
